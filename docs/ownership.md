# Owning Email Alert API

The owning team for this repo is responsible for the following maintenance tasks:

- Dependency Updates (as required)
- [Notify API key rotation][rotate-notify-keys]. Notify's guidance is that keys
  should be rotated once every six months, when someone with access to a key leaves
  the team, or whenever you suspect a key has been compromised. You will need to
  rotate the keys for the following services:
  - GOV.UK Email
  - GOV.UK Email (staging)
  - GOV.UK Email (integration)

[rotate-notify-keys]: https://docs.publishing.service.gov.uk/manual/govuk-notify.html#rotating-api-keys
